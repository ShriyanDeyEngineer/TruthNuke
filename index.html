Install Guide

Get up and running in under 2 minutes.

Option A — Download the ZIP

Easiest way to get started. No git or terminal needed.

  1. Click the Download Extension button above, or grab the latest truthnuke-extension.zip from GitHub Releases.
  2. Unzip the downloaded file. You'll get a folder called extension.
  3. Open Chrome and go to chrome://extensions/
  4. Turn on Developer mode using the toggle in the top-right corner.
  5. Click Load unpacked and select the unzipped extension folder.
  6. Done! Navigate to Twitter/X, Instagram, TikTok, or Facebook. TruthNuke badges will appear on financial posts.

Option B — Clone from GitHub

For developers who want the full source and backend.

  1. Clone the repository: 
    git clone https://github.com/nickcmiller/truthnuke.git
cd truthnuke
  2. Start the backend: 
    cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
# Add your ANTHROPIC_API_KEY to .env
python3 main.py
  3. Open Chrome → chrome://extensions/ → enable Developer mode.
  4. Click Load unpacked → select the extension/ folder from the cloned repo.
  5. The extension connects to http://localhost:8000 by default. You can change this in the extension popup settings.
